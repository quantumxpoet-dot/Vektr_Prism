Architecture
Browser (User's Machine)
├─ Vektr Prism (React App)
│  ├─ File Browser (local files via File System Access API)
│  ├─ Editor (Monaco)
│  └─ Chat Panel (clipboard bridge to AI)
└─ User's AI Chatbot (ChatGPT, Claude, etc.)

Zero backend. Zero API keys. Code never leaves user's machine.

Key Decisions Made
✅ Web-based (not desktop EXE)
✅ Static site (Cloudflare Pages, $0/mo)
✅ Clipboard bridge (copy-paste, no CDN compromise)
✅ File System Access API (read/write local files)
✅ Fully typed (React + File System API)
